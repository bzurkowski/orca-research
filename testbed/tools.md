# Tools

## Go

Download Go release:

```bash
$ wget https://dl.google.com/go/go1.14.2.linux-amd64.tar.gz
```

Extract the release:

```bash
$ tar -C /usr/local -xzf go1.14.2.linux-amd64.tar.gz
```

Add Go binary to system path:

```bash
$ echo "export PATH=\$PATH:/usr/local/go/bin" >> ~/.bash_profile
```
